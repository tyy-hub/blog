Js

let、const、var

  • let
    • 自带块级作用域，外部不可访问 {}代表块
    • 没有变量声明提升
    • 不允许重复声明(规范角度来说)
  • var
    • 外部可访问到
    • 可先使用后声明，此时值为 undefined
    • 可重复声明
  • const
    • 定义常量，不可修改
