数字

数字监测方法 typeof

  1. 数据类型

    • 字符串 string
    • 布尔型 boolean
    • 数字 number
    • undefined
    • null
    • 对象 object
    • symbol

  2. 字符串转换成数字

    • Number(str) 整体看能不能转换成数字，能就转，不能为 NaN

    • parseInt(str) 只能转换成整数 规则是一个字符一个字符的转换

    • parseFloat(str) 可以转换成小数 规则是一个字符一个字符的转换

    • （-，* 1，/，%）隐式转换

  3. 转换为字符串

    • String(num)
    • num.toString()

  4. 转换为布尔值

    • Boolean(str) 只有数字 0,空串,null,undefined,NaN,false 为 false ,其余为 true

  5. 隐式转换

    • (-，* 1，/，%)
    • 只要 + 的一边出现了字符串则拼接为字符串
    • 只要写在条件中自动转换为布尔型

字符串 String

  1. 属性

    • length 长度

  2. 方法

    • .charAt(index)获取对应位的字符

    • .includes(str) 判断当前字符串是否包含某个字符串，返回 true 或 false

    • .indexOf(str) 判断当前字符串是否包含某个字符串，返回开始位置或 -1 ||lastIndexOf(str)从后开始找

    • .trim()左右 .trimStart()左 .trimRight()右 清除字符串的空白(中间不去除)，原来字符串不变

    • .toUpparCase()转成大写 .toLowerCase()转成小写 (返回新的，原来字符串不变)

    • .split() 将字符串拆分成数组，返回新数组，原来字符串不变，对应数组中的 join()方法

    • .slice(a,b) 字符串的截取 包括 a 不包括 b，生成新的字符串

    • .repeat(n) 将字符串重复 n 次

    • .padEnd(length,str)向字符后面填充内容，填充到指定长度,str 省略默认填充空格 || .padStart()从前面填充

    • .search() 检测字符串中是否存在某个字符串，返回起始位置。括号内可写正则表达式，也可以写字符串，写字符串默认改变为对应的正则

    • .match()检测字符串中是否存在某个字符串，返回类数组(相当于用了正则的 exec 方法)。若正则带了 g，则会返回匹配到的所有字符串组成的数组。 || .matchAll()与 match()等同，但.matchAll()的正则表达式必须加 g

    • .replace()、.replaceAll() 将字符串中某些字符替换。逗号前可以是字符串也可以是正则,加 g 则为全局。如果用 replaceAll 那么正则表达式必须加 g

    • !== 判断不相等，会判断类型

  3. &&, ||, !

  4. ++, –, +=, -=, *=, /=

    • ++ (num++ 与++num 相同，只在输出时不同)

    • +=

  5. 三目简化 条件 ? (成立做的事 1,成立做的事 2) : (不成立做的事 1,不成立做的事 2)

数组 array

  • 数组的删除 splice pop shift
  • 数组的拷贝 slice map filter

  1. 数据的有序集合

  2. 属性 length 可以获得数组的长度(即元素个数)，如果赋值，大于原来长度就增加空值，如果小于原来长度，就截取。一般只是获取长度

  3. 方法

    • 数组的函数可以传三个值，分别为 element，index，array

    • Array.isArray(users); 检测 users 是否是数组

    • Array.from

    • Array.of()创建数组

    • arr.push(值) 或 arr.push(值，值，值，值，，，)在数组后面添加一个或者多个元素，并返回数组的新长度，修改原数组

    • arr.pop()删除数组的最后一个元素，并返回删除的元素，修改原数组

    • arr.unshift(值，值，值，，，)在数组的前面添加一个或多个元素，并返回数组的新长度，修改原数组

    • arr.shift()删除数组的第一个元素，并返回删除的元素，修改原数组

    • arr.splice(a,b,c1,c2)数组的删除或添加，在第 a 个位置开始的 b 个元素改成 c; c 可以省略,即只做删除不做添加; 返回值为被删除的数组，修改原数组

    • arr.split(‘’) 根据空格把一个字符串分割成字符串数组

    • arr.slice(a,b)截取数组，返回截取后的数组，不修改原数组；a：开始下标，如果 a 为负数代表倒数第几个开始；b：结束下标，如果省略 b 就截取到末尾。包括开始不包括结尾

    • arr.concat([1,2]) 函数 数组的拼接，返回拼接后的数组，原数组不变

    • arr.indexOf(值)监测值是否在数组中存在，如果存在返回位置，不存在返回-1…….当对象类型比较时比较的是地址而不是值

    • arr.includes()监测值是否在数组中存在，如果存在返回 true，不存在返回 false

    • arr.reverse() 颠倒数组顺序，返回颠倒后的数组，原数组也颠倒

    • user.sort(函数或者不写) 对数组进行排序，默认按照首字符从小到大进行排序，原数组改变

    • user.join(值)将数组使用该值拼接成字符串，返回新的字符串，原数组不变，默认用逗号拼接

    • user.forEach(函数)数组的遍历。依次查找数组内的元素，每次查找到执行 function 内的内容，function 可以设置三个参数：1.element 元素 2.index 索引 3.原数组

    • user.every(函数) 测试一个数组内的所有元素是否都满足 return 条件。它返回一个布尔值。如果为空数组则返回 true

    • find() 返回数组中满足条件(提供的测试函数)的第一个元素的值。找不到则返回 undefined

    • findIndex()方法返回数组中满足提供的测试函数的第一个元素的索引。若没有找到对应元素则返回-1。

    • filter(函数) 方法创建一个新数组, 返回满足条件的数组，若找不到则返回空数组，不改变原数组

    • map 方法创建一个新数组，其结果是该数组中的每个元素是调用一次提供的函数后的返回值。不改变原数组

    • some() 检测数组中是否存在满足条件(至少一个)的元素，返回 true 或 false

    • reduce() 方法接收两个参数。第一个是函数，该函数可以接收四个参数:1.最终结果(累加器)2，3，4 分别为 element，index，array，该函数必须设置一个返回值，这个返回值是新的最终结果。第二个参数是最终结果的初始值，想要拿到什么就设置对应的初始值

日期 Date

  1. 创建日期对象
    • var date = new Date()
  2. 方法
    • date.getFullYear()获取年份
    • date.getMonth()获取月份,,,,月为 0 到 11
    • date.getDate()获取日份
    • date.getDay()获取星期，默认为 1 到 0，0 为周日
    • date.getHours()获取小时
    • date.getMinutes()获取分钟
    • date.getSeconds()获取秒数
    • date.setMilliseconds()获取毫秒
    • date.getTime()获取格林威治时间
    • 上述方法 get 后加上 UTC 获取的就是世界时间
    • 将 get 换成 set 的话就是设置时间

正则表达式 RegExp

  1. 其实就是一个规则 就是用来检测字符串或者数字的规则

  2. 方法

    • test 普通检测 返回 true 或 false
    • exec 高级检测 返回结果对象数组（第一项为满足正则的部分，可以用[0]获取），若不符合正则就返回 null

  3. 规则

    • ^设置开头

    • $设置结尾

    • [a-z],[A-Z] 一位字母 | [A-z]检测一位字符或下划线

    • [0-9] 一位数字，用法同上

    • {数字} 代表多少位 ||| {10} 10 位 || {1,} 至少一位，可简写成 + || {0,1} 没有或有一位，可简写成 ? || {2,5} 2-5 位 || {0,} 有没有都可以，可简写成 *

    • | 代表或

    • () 分组

    • ()\1 对括号内的内容重复

    • \d 代表一位数字 || \D 代表一位非数字 || \w 代表一位数字，字母，下划线 || \W 代表一位非数字，字母，下划线 || \s 代表一位空白符(空格，tab) || \S 代表一位非空白符(空格，tab)

    • /a/i 中 i 表示不区分大小写

    • [^ab]一位非小写字母 a b 的字符

    • (?=b)检测是否紧跟着 b

    • . 代表任意字符

    • \代表匹配正则内的运算符也可以是^,$

    • [\u4e00-\u9fa5]一位汉字，简体中文

    • 变量正则

  4. 标志参数

    • i 不区分大小写
    • g 全局匹配
  • 判断重复

同步异步操作

  • 同步操作按顺序依次执行

  • 异步执行和其他代码无关

  • 异步在同步执行完后执行

浏览器对象 window

  1. 方法

    • setInterval 每 1000 毫秒一秒运行一次 getTime 函数

    • 停止计时器 clearInterval 清除 res 的 setInterval

    • setTimeout 设置代码 1000 毫秒后运行

    • clearTimeout 停止 setTimeout

    • open 打开新的一个窗口 open(“地址”,”新打开的窗口名称”,”窗口的大小宽高等”)

    • close 关闭当前窗口（一般不用）

    • innerheight innerwidth 返回窗口大小

    • outheight outwidth 返回屏幕大小

  2. 弹窗类

    • alert 弹出一个弹窗只有一个确定按钮
    • confirm 弹出一个弹窗有一个确定按钮和一个取消按钮，可以给一个返回值，确定为 true 取消为 false
    • prompt 弹出一个弹窗有有一个输入框、一个确定按钮和一个取消按钮可以括号内写两个值，第二个为输入框默认值，可以给一个返回值，确定为输入内容，取消为 null。

  3. 事件

    • onscroll 原生的页面滚动条事件

  4. location 可返回页面地址的一些信息

    • href 获取当前页面的地址
    • hash 获取页面一些锚点的信息也可以修改
    • host 获取主机(域名)和端口号(如果后面带了)
    • hostname 获取主机(域名)
    • pathname 获取路径部分
    • search 获取或者设置查询部分
    • assign 跳转 有历史记录
    • replace 替换 无历史记录
    • reload 刷新

  5. history 浏览器历史记录 也是一个对象

    • length 可返回次数
    • back 后退
    • go 前进或后退 负数为后退 正数是前进 0 是刷新
    • forward 前进
