  2. жңҖеҲқи®ҫи®Ў

    • null иҪ¬дёәж•°еҖјж—¶дёә 0
    • undefined иҪ¬дёәж•°еҖјж—¶дёә NaN

  3. зӣ®еүҚз”Ёжі•

    • null иЎЁзӨәвҖқжІЎжңүеҜ№иұЎвҖқпјҢеҚіиҜҘеӨ„дёҚеә”иҜҘжңүеҖјпјҢдёҚеҚ еҶ…еӯҳгҖӮе…ёеһӢз”Ёжі•жҳҜпјҡ

      • дҪңдёәеҜ№иұЎеҺҹеһӢй“ҫзҡ„з»ҲзӮ№

    • undefined иЎЁзӨәвҖқзјәе°‘еҖјвҖқпјҢе°ұжҳҜжӯӨеӨ„еә”иҜҘжңүдёҖдёӘеҖјпјҢдҪҶжҳҜиҝҳжІЎжңүе®ҡд№үпјҢеҚ еҶ…еӯҳгҖӮе…ёеһӢз”Ёжі•жҳҜпјҡ

      • еЈ°жҳҺжңӘиөӢеҖј

      • и°ғз”ЁеҮҪж•°ж—¶пјҢеә”иҜҘжҸҗдҫӣзҡ„еҸӮж•°жІЎжңүжҸҗдҫӣпјҢиҜҘеҸӮж•°зӯүдәҺ undefinedгҖӮ

      • еҮҪж•°жІЎжңүиҝ”еӣһеҖјж—¶пјҢй»ҳи®Өиҝ”еӣһ undefinedгҖӮ

      6. е°ҶеҠЁз”»ж•Ҳжһңеә”з”ЁеҲ° position еұһжҖ§дёә absolute жҲ– fixed зҡ„е…ғзҙ дёҠпјҢйҒҝе…ҚеҪұе“Қе…¶д»–е…ғзҙ зҡ„еёғеұҖпјҢиҝҷж ·еҸӘжҳҜдёҖдёӘйҮҚз»ҳпјҢиҖҢдёҚжҳҜеӣһжөҒпјҢеҗҢж—¶пјҢжҺ§еҲ¶еҠЁз”»йҖҹеәҰеҸҜд»ҘйҖүжӢ© requestAnimationFrameгҖӮ

      8. е°Ҷйў‘з№ҒйҮҚз»ҳжҲ–иҖ…еӣһжөҒзҡ„иҠӮзӮ№и®ҫзҪ®дёәеӣҫеұӮпјҢеӣҫеұӮиғҪеӨҹйҳ»жӯўиҜҘиҠӮзӮ№зҡ„жёІжҹ“иЎҢдёәеҪұе“ҚеҲ«зҡ„иҠӮзӮ№пјҢдҫӢеҰӮ will-changeгҖҒvideoгҖҒiframe зӯүж ҮзӯҫпјҢжөҸи§ҲеҷЁдјҡиҮӘеҠЁе°ҶиҜҘиҠӮзӮ№еҸҳдёәеӣҫеұӮгҖӮ

дәӢд»¶е§”жүҳ

  • е®ҳж–№и§ЈйҮҠ
    дәӢд»¶е§”жүҳд№ҹеҸ«дәӢд»¶д»ЈзҗҶ, javaScript й«ҳзә§зЁӢеәҸи®ҫи®ЎдёҠи®ІпјҡдәӢд»¶е§”жүҳе°ұжҳҜеҲ©з”ЁдәӢд»¶еҶ’жіЎпјҢеҸӘжҢҮе®ҡдёҖдёӘдәӢд»¶еӨ„зҗҶзЁӢеәҸпјҢе°ұеҸҜд»Ҙз®ЎзҗҶжҹҗдёҖзұ»еһӢзҡ„жүҖжңүдәӢд»¶гҖӮ(еҫҲе®ҳж–№)

  • з®ҖеҚ•зҗҶи§Јпјҡ
    еҚіжҳҜжҠҠеҺҹжң¬йңҖиҰҒз»‘е®ҡеңЁеӯҗе…ғзҙ зҡ„е“Қеә”дәӢд»¶(clickгҖҒkeydownвҖҰ)е§”жүҳз»ҷзҲ¶е…ғзҙ пјҢи®©зҲ¶е…ғзҙ жӢ…еҪ“дәӢд»¶зӣ‘еҗ¬зҡ„иҒҢеҠЎгҖӮдәӢд»¶д»ЈзҗҶзҡ„еҺҹзҗҶжҳҜ DOM е…ғзҙ зҡ„дәӢд»¶еҶ’жіЎгҖӮ

  • дәӢд»¶е§”жүҳзҡ„дҪңз”Ёпјҡ

    1. з»‘е®ҡзҡ„дәӢд»¶и¶ҠеӨҡпјҢжөҸи§ҲеҷЁеҶ…еӯҳеҚ з”Ёи¶ҠеӨ§пјҢдёҘйҮҚеҪұе“ҚжҖ§иғҪгҖӮ
    2. ajax зҡ„еҮәзҺ°пјҢеұҖйғЁеҲ·ж–°зҡ„зӣӣиЎҢпјҢеҜјиҮҙжҜҸж¬ЎеҠ иҪҪе®ҢпјҢйғҪиҰҒйҮҚж–°з»‘е®ҡдәӢд»¶гҖӮ
    3. йғЁеҲҶжөҸи§ҲеҷЁз§»йҷӨе…ғзҙ ж—¶пјҢз»‘е®ҡзҡ„дәӢд»¶е№¶жІЎжңүиў«еҸҠж—¶з§»йҷӨпјҢеҜјиҮҙзҡ„еҶ…еӯҳжі„жјҸпјҢдёҘйҮҚеҪұе“ҚжҖ§иғҪгҖӮ
    4. еӨ§йғЁеҲҶ ajax еұҖйғЁеҲ·ж–°зҡ„еҸӘжҳҜжҳҫзӨәзҡ„ж•°жҚ®пјҢиҖҢж“ҚдҪңеҚҙжҳҜеӨ§йғЁеҲҶзӣёеҗҢзҡ„пјҢйҮҚеӨҚз»‘е®ҡдјҡеҜјиҮҙд»Јз Ғзҡ„иҖҰеҗҲжҖ§иҝҮеӨ§пјҢдёҘйҮҚеҪұе“ҚеҗҺжңҹзҡ„з»ҙжҠӨгҖӮ

  • дјҳзӮ№пјҡ

    1. еҸҜд»ҘеӨ§йҮҸиҠӮзңҒеҶ…еӯҳеҚ з”Ё
    2. еҠЁжҖҒз”ҹжҲҗзҡ„ DOM д№ҹеҸҜд»Ҙи§ҰеҸ‘дәӢд»¶
